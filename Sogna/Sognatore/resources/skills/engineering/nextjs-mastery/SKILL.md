# Next.js & React Mastery

Directrices avanzadas para el desarrollo con el App Router de Next.js y los paradigmas modernos de React.

---

## ⚡ 1. React Server Components (RSC)
*   **Por Defecto**: Mantener los componentes como Server Components siempre que sea posible.
*   **Componentes de Cliente**: Usar la directiva `'use client'` únicamente para interactividad, hooks de estado o eventos del navegador.

## 🌐 2. Estrategias de Carga de Datos
*   **Server Actions**: Emplear Server Actions para mutaciones de datos de manera segura y robusta.
*   **Fetching**: Realizar peticiones directamente en los Server Components mediante `async/await`.

## 🎨 3. Optimización y UX
*   **Streaming y Suspense**: Diseñar interfaces con estados de carga granulares.
*   **Core Web Vitals**: Implementación de imágenes y fuentes optimizadas nativas.
