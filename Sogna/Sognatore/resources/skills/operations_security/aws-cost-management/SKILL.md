# AWS Cost Management & Optimization

Estrategias operativas para el control de presupuestos y limpieza de recursos en Amazon Web Services.

---

## 📊 1. Monitoreo y Optimización

* **AWS Cost Explorer**: Detección de anomalías y picos de uso.
* **Recomendaciones de Compute Optimizer**: Ajuste del tamaño (rightsizing) de instancias EC2 y Lambda.

## 🧹 2. Limpieza de Recursos Obsoletos

* **Volúmenes EBS Huérfanos**: Protocolo para identificar y borrar volúmenes no vinculados.
* **Snapshots Antiguos**: Políticas de retención automatizadas.

---
*Nota: Fusión operativa de capacidades Sentinel.*
