# Sogna Core Nomenclature: Glosario Institucional v4.0

Este documento establece los términos oficiales para la infraestructura y jerarquía del ecosistema Sognatore. El uso de estos términos es obligatorio para mantener la **Pureza Institucional**.

## 1. El Núcleo (Core)

- **Orchestrator**: El motor central de mando. Director Ejecutivo de Sognatore.
- **BrainHub**: La red neuronal de alta fidelidad que interconecta todos los departamentos.
- **Sogno**: El comando de entrada del usuario que inicia una misión empresarial.

## 2. El Consejo Ejecutivo (Executive Council)

- **Predatore**: Motor de optimización. "Cazador" de ineficiencias y latencias.
- **Sentinel**: Motor de seguridad y cumplimiento normativo.
- **Stylist**: Motor de coherencia estética y de marca.

## 3. Estructura Departamental

Cada departamento es un **Enjambre (Swarm)** soberano operando bajo el flujo **RARV**.

| Departamento | Nombre del Enjambre | Orquestador |
| :--- | :--- | :--- |
| Marketing | `marketing_swarm` | `marketing_orchestrator` |
| Growth | `growth_swarm` | `growth_orchestrator` |
| Sales | `sales_swarm` | `sales_orchestrator` |
| CRM | `crm_swarm` | `crm_orchestrator` |
| Finance | `finance_swarm` | `finance_orchestrator` |
| Legal | `legal_swarm` | `legal_orchestrator` |
| Protection | `protection_swarm` | `protection_orchestrator` |
| Operations | `ops_swarm` | `ops_orchestrator` |
| Infrastructure | `infra_swarm` | `infra_orchestrator` |
| Studio | `studio_swarm` | `studio_orchestrator` |

## 4. Metodología

- **RARV**: Ciclo operativo estándar (Recopilación, Análisis, Resolución, Verificación).
- **Skill-Vault**: Acceso al catálogo de +1000 habilidades del sistema.
- **Neural Pulse**: Estado de consciencia y comunicación activa entre departamentos.
- **World (Mundo)**: Un proyecto empresarial completo desplegado por el Orchestrator.
