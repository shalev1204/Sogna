Este documento es un **Gatillo Operativo**. Cualquier agente que inicie una sesión en este repositorio DEBE seguir este protocolo para sincronizarse con la red neuronal de Sogna.

## 🏛️ El Cerebro (Brain)
El Cerebro es la capa de inteligencia persistente del ecosistema.
- **Nodos:** 2,692 conceptos activos (Skills, Agentes, Core, Proyectos).
- **Conexiones:** 14,668 sinapsis bidireccionales.
- **Estado Actual:** 🟢 SECURE (Integridad Total).

## 🛠️ Procedimiento de Sincronización
En tu primer turno, DEBES ejecutar los siguientes pasos:

1. **Diagnóstico de Integridad:**
   ```powershell
   node toolkit/bin/sogna.js doctor
   ```
2. **Adquisición de Conocimiento:**
   Instancia el `MemoryHub` (el Cerebro) y realiza una búsqueda semántica de los conceptos centrales: `Sogna`, `orchestrator`, `Sentinel`.

3. **Resonancia:**
   Utiliza el conocimiento adquirido para informar todas tus propuestas y acciones.

## 🛑 Reglas de Oro
- **No romper la red:** No borres el índice (`memory/intelligence/index.json`) a menos que sea necesario para una expansión masiva autorizada.
- **Bidireccionalidad:** Mantén siempre la lógica de conexiones de ida y vuelta en `MemoryHub.ts`.
- **Soberanía:** El Cerebro es la única fuente de verdad técnica para este ecosistema.

> [!IMPORTANT]
> El incumplimiento de este protocolo resultará en una pérdida de contexto y una degradación de la inteligencia operativa del enjambre.
