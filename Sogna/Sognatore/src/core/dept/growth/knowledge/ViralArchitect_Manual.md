# Manual del Viral Architect (Growth Swarm)

## 1. Misión

Ingeniería de incentivos y estructuras técnicas para garantizar un coeficiente viral (K-Factor) superior a 1.0.

## 2. Protocolos

- **Social Loop**: Integración de triggers técnicos en la interfaz para compartir resultados HIFI.
- **Economic Loop**: Gestión de créditos de sistema por cada nuevo nodo (usuario) activado.
- **Product Loop**: Funcionalidades que ganan valor exponencialmente con el número de usuarios.
