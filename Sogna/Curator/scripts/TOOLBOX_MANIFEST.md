# laboratorio sogna: manifiesto de herramientas (estado: base limpia)

este es el inventario de herramientas maestras en el laboratorio de experimentación (scratch). el laboratorio ha sido purgado de pruebas pasadas para garantizar un entorno de trabajo puro y optimizado.

## 🚀 herramientas maestras (toolbox/)

* **sogna_lab.py**: la navaja suiza de auditoría y reparación. consolida la lógica de auditoría de navigator, reparación de código mediante regex y baterías de tests lógicos. admite parámetros `--mode [audit|repair|test]`.
* **md_purifier.py**: el brazo ejecutor de la estética. script de purificación masiva que aplica el protocolo de gobernanza a todos los archivos .md del proyecto.
* **SognaSanitizer.py**: motor de normalización de nomenclatura (lowercase) y reparación de importaciones.
* **unify_react.js**: controlador de dependencias críticas para la coherencia de versiones en el monorepo.

## 🧪 incubadora de experimentos (labs/)

esta carpeta se encuentra actualmente **limpia**. es el espacio designado para nuevas investigaciones, validaciones de hipotesis y prototipado rápido.

---

## 🛡️ protocolos de seguridad

el entorno está monitorizado proactivamente por **veglia.js**, que ejecuta tareas de sanitización y purificación de forma autónoma en cada ciclo de trabajo.
