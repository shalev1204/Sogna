## Why sognaflow?

-   **Simple API:** First-class React, JavaScript, and Vue packages.
-   **Hybrid engine:** Power of JavaScript combined with native browser APIs for 120fps, GPU-accelerated animations.
-   **Production-ready:** TypeScript, extensive test suite, tree-shakable, tiny footprint.
    **Batteries included:** Gestures, springs, layout transitions, scroll-linked effects, timelines.

## 🍦 Platforms

sognaflow is available for [React](https://sognaflow.dev/docs/react), [JavaScript](https://sognaflow.dev/docs/quick-start) and [Vue](https://sognaflow.dev/docs/vue).

### React

```jsx
import { sognaflow } from "sognaflow/react"

function Component() {
    return <sognaflow.div animate={{ x: 100 }} />
}
```

Get started with [sognaflow for React](https://sognaflow.dev/docs/react).

**Note:** Framer sognaflow is now sognaflow. Import from `sognaflow/react` instead of `framer-sognaflow`.

### JS

```javascript
import { animate } from "sognaflow"

animate("#box", { x: 100 })
```

Get started with [JavaScript](https://sognaflow.dev/docs/quick-start).

### Vue

```html
<script>
    import { sognaflow } from "sognaflow-v"
</script>

<template> <sognaflow.div :animate={{ x: 100 }} /> </template>
```

Get started with [sognaflow for Vue](https://sognaflow.dev/docs/vue).
