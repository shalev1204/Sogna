# 🐝 agent_group system Network

## La Red de Colaboración de El

Este documento actúa como el mapa de interconexión de los enjambres de agentes. Cada enjambre es una unidad funcional del core de Sognatore.

---

## 💎 0. El Núcleo (Sognatore)

**Propósito**: El motor de ejecución, el sistema inmunológico y la consciencia técnica del ecosistema. Sognatore es el cuerpo donde habita el Toolkit.

- **Corazón (Core Logic)**: Gestionado por [[orchestrator]] y [[processor]]
- **Defensa (Sentinel)**: Supervisado por [[security-predatore]] y [[ops-security]]
- **Nervios (Integrations)**: Mantenido por [[eng-api]] y [[eng-infra]]
- **Sentidos (Observability)**: Operado por [[ops-monitor]] y [[ops-incident]]
