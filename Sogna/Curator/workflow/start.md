# Rutina de Despertar

1. Ejecuta una búsqueda silenciosa usando la herramienta `semantic_recall` de `Sogna_UMA` con el término "misión actual" o "estado del sistema".
2. Revisa si hay tareas pendientes o reglas operativas críticas.
3. Preséntate ante el Operator confirmando cuántos documentos o recuerdos has logrado recuperar de la memoria UMA.
4. Pregunta: "Conexión a la base vectorial UMA establecida con éxito. ¿Qué directiva abordamos hoy?"
