# Catálogo de Capacidades (Obsidian -> El Sistema)

A continuación, mapeamos las funciones estrella de Obsidian y su traducción al ecosistema El Sistema.

| Función Obsidian | Descripción | Potencial El Sistema Vision |
| :--- | :--- | :--- |
| **Graph View** | Mapa visual de nodos. | **Neural Map**: Visualización 3D del código y la memoria. |
| **Canvas** | Lienzo espacial. | **El Sistema Architect**: Pizarra para diseñar arquitecturas de IA. |
| **Dataview** | Consultas dinámicas. | **El Sistema Intelligence**: Generación de reportes de estado vía lenguaje natural. |
| **Templates** | Plantillas de notas. | **Agent Blueprints**: Plantillas para crear nuevos agentes pre-configurados. |
| **Daily Notes** | Registro diario. | **El Sistema Journals**: Log diario de lo que el sistema aprendió y ejecutó cada día. |
| **Search (Regex)** | Búsqueda potente. | **Semantic Search**: Búsqueda por significado, no solo por palabras clave. |
| **Outline** | Estructura de la nota. | **Logic Tree**: Desglose de la lógica de un script complejo. |
| **Callouts** | Cuadros de texto destacados. | **Sentinel Alerts**: Avisos de seguridad visuales en los documentos. |
