# Plugins de Poder: Automatización y Control Total

Hemos analizado los tres pilares de la automatización avanzada en Obsidian para replicar su potencia de forma nativa.

## 1. Templater (JS Contextual)
Templater permite ejecutar código Javascript dentro de las notas, accediendo a la API interna de Obsidian.
- **Nuestra Estrategia**: "Active Templates". No solo plantillas estáticas, sino archivos de "instrucciones vivas" que los agentes pueden ejecutar para auto-configurar el entorno de trabajo.

## 2. Dataview JS (Database-as-a-Vault)
Dataview JS permite usar `dv.query()` para renderizar datos complejos.
- **Nuestra Estrategia**: Un "Intelligence Dashboard" nativo en el toolkit que use lógica de JavaScript para visualizar el estado del proyecto, el inventario de skills y la carga de trabajo de los agentes sin necesidad de plugins externos.

## 3. Advanced URI (The Bridge)
Permite la comunicación entre aplicaciones mediante protocolos URI.
- **Nuestra Estrategia**: Implementar un sistema de "Deep Links" que permita invocar agentes o abrir vistas específicas del visualizador desde cualquier parte del sistema operativo o desde aplicaciones web externas.
