## technical protocols (mandatory)
### lexicon & tone
- **tone**: strictly professional, technical, and direct.
- **prohibited**: no grandiloquent adjectives (sovereign, apex, supreme, elite, divine, etc.).
- **currency**: euro (€) is the institutional standard for all financial data.

### development standards
- **language**: typescript (esm) for all core modules.
- **workflow**: plan-first approach. every non-trivial task requires an `implementation_plan.md`.
- **pipeline**: scripts-first protocol. all modules must expose `build`, `check`, and `lint`.
- **stability**: run `npm run check` (global) or `node Sogna/Curator/bin/sogna.js doctor` before final delivery.

### ui/ux excellence
- **aesthetic**: modern, vibrant, glassmorphism, dynamic animations.
- **standard**: no placeholders. use functional assets and premium typography (inter/roboto).
- **feedback**: mandatory loading/error/success states for all components.
